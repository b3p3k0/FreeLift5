# FreeLift5 v0.1.0

First public APK release of FreeLift5, an offline Android tracker for classic
5x5 training.

## Highlights

- Multiple built-in training programs: Original 5x5, Lite, Mini, Plus, and
  5xOTG.
- Workout tracking with persistent active sessions, rest timers, progression,
  deload suggestions, barbell warmups, plate loading, history, charts, and PRs.
- Program switching without losing workout history.
- Optional on-device reminders and background timer alerts.
- Versioned CSV and ZIP export.
- Five Material 3 color themes with system-following or fixed appearance:
  Solarized Light, Field Manual, Pool Tile, Solarized Dark, and Foundry.

## Privacy

- No account.
- No telemetry.
- No ads.
- No network permission.
- No automatic Android backup.
- Data stays local unless you explicitly export it.

## Install Notes

Download `FreeLift5-v0.1.0.apk` from the release assets and install it on
Android 9 / API 28 or newer.

If a debug build is already installed, Android may reject this release APK
because debug and release builds use different signing certificates. Uninstall
the debug build first, then install this release APK.
