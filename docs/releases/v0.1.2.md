# FreeLift5 v0.1.2

Patch release for version-label continuity, About links, workout polish, and
clearer Progress charts.

## Fixes

- Settings About now reads the Android build version instead of a hardcoded
  string, so the displayed app version follows release metadata.
- Settings About now links to the GitHub repository and GPL-3.0 full text by
  handing public URLs to the user's browser.
- Rest timer completion now fires a foreground cue when the Workout screen is
  visible and uses a real border pulse instead of a static border change.
- Completing the final set of a workout now clears the rest timer instead of
  starting a new rest interval.
- Workout set markers stay blank until a set is saved, then show recorded reps
  with a filled background.
- Progress charts now show compact date and value axes with metric-aware units.
- README download links now point to the v0.1.2 APK asset.

## Install Notes

Download `FreeLift5-v0.1.2.apk` from the release assets and install it on
Android 9 / API 28 or newer.

If a debug build is already installed, Android may reject this release APK
because debug and release builds use different signing certificates. Uninstall
the debug build first, then install this release APK.
