# FreeLift5 v0.1.1

Hotfix release for the first public APK.

## Fixes

- Keeps onboarding Back and Next controls above Android gesture and three-button
  navigation bars on edge-to-edge devices.

## Install Notes

Download `FreeLift5-v0.1.1.apk` from the release assets and install it on
Android 9 / API 28 or newer.

If a debug build is already installed, Android may reject this release APK
because debug and release builds use different signing certificates. Uninstall
the debug build first, then install this release APK.
